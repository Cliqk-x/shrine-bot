# ⛩️ SHRINE 神社 Bot

A cinematic Discord RPG bot built with Node.js and discord.js.  
Focused on progression, rituals, and interactive gameplay systems.

---

## ✨ Features

- 🎮 XP & Leveling System  
- 🔮 Ritual System (interactive + risk/reward)  
- ⏱️ Cooldown & Anti-Spam Protection  
- 🧩 Modular Architecture  
- ⚔️ Moderation Commands  

---

## 🎬 Highlight

The bot includes a **cinematic ritual system** where users:
- Experience suspense phases  
- Choose ritual types  
- Face risk vs reward outcomes
